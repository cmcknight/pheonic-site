<!-- Excerpt Start-->
The PyDevMgr provides a simple GUI interface to launch development tools/environments.<!-- Excerpt End -->

### Features

* Create new task
* Execute task
* Update task
* Track number of times executed
* Remove task
* Sorts tasks by number of times executed in descending order
